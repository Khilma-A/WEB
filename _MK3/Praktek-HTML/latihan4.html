Gaya Belajar di Rumah

Biasanya saya belajar melalui berbagai media baik itu handphone, laptop, buku, dan lainnya. Saya suka menghabiskan waktu belajar di rumah, karena rumah saya memiliki suasana sunyi sehingga saya merasa lebih fokus untuk belajar. Biasanya saya belajar setelah pulang sekolah. Sebelum belajar saya terlebih dahulu membuat skala prioritas mengenai mata pelajaran apa yang harus dipelajari terlebih dahulu, barulah setelah itu saya mulai mempelajarinya.


Kendala Selama Belajar di Rumah

Terkait kendala dalam belajar, sering kali saya mengalami berbagai kendala. Waktu menjadi kendala terbesar bagi saya dalam belajar, sebagian besar waktu yang saya miliki sudah dihabiskan untuk bebagai kegiatan lain, seperti bersekolah, berorganisasi, dan lain-lain. Sehingga sering kali saya tidur larut untuk belajar dan menyelesaikan tugas. Selain itu ada pula kendala lain seperti kurangnya bantuan dalam belajar. Walaupun sekarang teknologi sudah berkembang, namun ada kalanya saya membutuhkan orang lain untuk memberikan penjelasan kepada saya mengenai materi yang saya pelajari secara langsung, agar saya lebih memahami materi.
